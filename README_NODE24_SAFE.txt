# ملاحظات مهمة قبل الدمج

تم تعديل package.json داخل هذا الملف ليستخدم:
"engines": { "node": "24.x" }

لا تنقل هذا المشروع فوق مشروعك مباشرة إلا بعد أخذ نسخة احتياطية أو العمل على branch جديد.

الخطوات الآمنة:
1) اعمل Commit للنسخة الحالية قبل الدمج.
2) انسخ ملفات src و public و ملفات SQL المطلوبة.
3) لا تنس تشغيل CUSTOMER_CODING_SETUP.sql في Supabase بعد مراجعتها.
4) بعد الدمج شغل npm run build أو ارفع على Vercel واعمل Redeploy بدون Cache.
