# نموذج الصلاحيات التشغيلي — Dawaa Pharmacy

تم تثبيت قاعدة مهمة في التطبيق: أي صلاحية مخصصة لا يمكنها أن تفتح صفحة خارج نطاق دور الموظف. يمكن للصلاحيات المخصصة أن تغلق صلاحية من الدور، لكنها لا ترفع الموظف لمستوى أعلى من دوره.

## خريطة الأدوار والصفحات المسموحة

| الدور | نطاق البيانات | الصفحات الأساسية التي تظهر |
|---|---|---|
| مدير عام | كل الفروع | كل النظام، الحسابات، الصلاحيات، التقارير، صحة البيانات، مركز القيادة |
| مدير تنفيذي | كل الفروع | مركز القيادة، المبيعات، العملاء، الفريق، التقارير، المخزون، التقييمات |
| مدير الفروع | كل الفروع | أداء الفروع، المبيعات، الفريق، الشيفتات، العملاء، المخزون، التقارير |
| مدير المشتريات | كل الفروع | المخزون، النواقص، المشتريات، الموردين، الصلاحية، التقارير التشغيلية |
| مدير فرع | فرعه فقط | داشبورد الفرع، اليوم في لمحة، الفريق، الجدول، العملاء، التوصيل، النقاط، التقييمات |
| مدير خدمة العملاء | فرعه فقط | خدمة العملاء، المتابعات، CRM، حضانة العملاء، الكاشباك، تقييمات المحادثات |
| مشرف شيفت | فرعه فقط | اليوم في لمحة، الشيفت، الفريق الموجود، متابعات أساسية، نقاط محدودة |
| صيدلي | فرعه فقط | صفحته الشخصية، العملاء، متابعة العميل، تقييم محادثة، الأدوية الراكدة والحوافز |
| مساعد مخزون | فرعه فقط | المخزون، النواقص، الأدوية، الصلاحية، المشتريات للعرض |
| خدمة عملاء | فرعه فقط | العملاء، المتابعات، واتساب، CRM، حضانة العملاء، تقييم محادثة |
| دليفري | المسند له فقط | التوصيل وطلبات التوصيل فقط |
| مساعد / نظافة | محدود | داشبورد مبسط فقط |

## قاعدة الحماية الجديدة

- مدير عام فقط يحصل على كل الصلاحيات.
- أي موظف آخر يحصل على صلاحيات دوره فقط.
- لو قاعدة البيانات فيها صلاحية زائدة بالخطأ مثل `manage_permissions` لموظف خدمة عملاء، التطبيق سيتجاهلها لأنها خارج نطاق الدور.
- يمكن تعطيل صلاحية من داخل نفس الدور، لكن لا يمكن منح صلاحية أعلى من الدور.

## ملف استخراج تقرير الموظفين

استخدم الملف التالي داخل Supabase SQL Editor لاستخراج تقرير الموظفين والحسابات والصلاحيات:

`SUPABASE_STAFF_PERMISSION_AUDIT_REPORT.sql`

الناتج المطلوب منه:
- اسم الموظف
- اسم المستخدم
- الدور
- الفرع
- المسؤولية
- نطاق البيانات
- الصفحات المسموحة
- عدد الصلاحيات الفعالة
- ملاحظات الصلاحيات الزائدة أو الحسابات غير المربوطة

## ملاحظات تشغيلية

بعد أي تغيير في الصلاحيات، يجب تسجيل الخروج والدخول مرة أخرى حتى يتم تحميل الصلاحيات الجديدة من جديد.
