# إصلاح نطاق المسابقة وتقييمات الدكتور والإشعارات

- صفحة مسابقة الدكاترة وحدها تعرض جميع الدكاترة وتستكمل القائمة من ملخص المبيعات حتى لو كانت RLS على حسابات الموظفين تعيد الحساب الحالي فقط.
- باقي صفحات الدكتور تظل شخصية ومقيدة بالحساب الحالي.
- التقييمات القديمة تُربط تلقائيًا بـ `staff_id` عند وجود تطابق وحيد للاسم والفرع.
- التقييمات الجديدة تحصل على `staff_id` قبل الحفظ.
- إشعار التقييم يكتب الحقول الحديثة والحقول التي يقرأها صندوق الجرس العلوي معًا.
- يتم إنشاء إشعار مراجعة للتقييمات القديمة التي تم ربطها.
