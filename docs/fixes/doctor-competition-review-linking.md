# إصلاح تقييمات مسابقة الدكاترة

- إعادة تحميل تقييمات المحادثات للفترة المختارة من `conversation_sales_reviews`.
- المطابقة بالترتيب: `staff_id` ثم `doctor_id` ثم الاسم الموحد مع الفرع ثم الاسم الفريد.
- استخدام `final_score` ثم `total_score` كبديل.
- استبدال أرقام التقييم في الصف النهائي لمنع الجمع المزدوج.
- إظهار متوسط التقييم، عدد التقييمات، والمتابعات المكتملة داخل جدول المسابقة.
